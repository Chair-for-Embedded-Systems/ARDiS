# Examples

This folder contains various examples to help you understand how to use `ARDIS`.

## How to run
Assuming the virtual environment is properly setup and you are `root`, all examples can be run by simpling executing the following command:
``` bash 
python3 <example>.py
```

**Without virtual environment (not recommended)**

In case you are not using the virtual environment, you must point the `PYTHONPATH` environment variable the `ardis` directory:
``` bash
sudo PYTHONPATH=/path/to/ardis/ python3 <example>.py
```
